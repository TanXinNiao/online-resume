# online-resume是一个基于react和gin实现的前后端分离项目

## 先启动后端项目gin-resume（注意启动前需把go模块全导入）

```
cd gin-resume
go run main
```

## 先启动前端项目react-resume

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```
